ECommerce application built with Gatsby, React, Tailwind, Netlify and Netlify CMS


## Getting started

1. Clone the project

```shell
$ git clone https://github.com/gustavomonjardim/ecommerce.git
```

2. Install the dependencies:

```shell
$ yarn
```

3. Run the project

```shell
$ yarn develop

# or to build

$ yarn build
```

## Testing

```shell
$ yarn test
```
